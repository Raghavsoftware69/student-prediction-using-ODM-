# 🎓 Student Performance Prediction using Orange Data Mining (ODM)

## 📖 Description

Student Performance Prediction using Orange Data Mining (ODM) is a machine learning project that predicts whether a student is likely to **Pass** or **Fail** based on important academic factors such as **Study Hours**, **Attendance**, and **Previous Marks**. The project demonstrates how machine learning can be applied in the education sector to analyze student data, identify learning patterns, and make accurate predictions that help improve academic performance.

The project is developed using **Orange Data Mining (ODM)**, an open-source visual machine learning and data mining platform. Orange provides a drag-and-drop interface that enables users to build complete machine learning workflows without writing complex code. This makes it an excellent tool for beginners, students, educators, and researchers who want to understand machine learning concepts through visual programming.

The workflow begins by loading a dataset containing student information. After importing the data, preprocessing techniques are applied to clean and organize the dataset for analysis. Various classification algorithms, including **Decision Tree**, **Random Forest**, **Logistic Regression**, **Naïve Bayes**, **K-Nearest Neighbors (KNN)**, and **Support Vector Machine (SVM)**, can be used to train predictive models. The performance of each model is evaluated using standard metrics such as **Accuracy, Precision, Recall, F1-Score, AUC (Area Under the Curve)**, and **Confusion Matrix** to determine the best-performing classifier.

The primary objective of this project is to demonstrate how educational institutions can use machine learning to identify students who may require additional academic support before final examinations. Early prediction enables teachers and administrators to take proactive measures such as personalized mentoring, remedial classes, and continuous performance monitoring, ultimately improving student success rates.

This repository includes the complete Orange workflow (`.ows`), dataset (`.csv`), Python implementation (if applicable), screenshots of the workflow and prediction results, and detailed documentation. It serves as an academic learning resource for **BCA, MCA, B.Tech, Computer Science, Data Science, Artificial Intelligence, and Machine Learning** students who want to explore visual machine learning using Orange Data Mining.
